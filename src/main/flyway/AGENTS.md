# Specific agent instructions for migration scripts SQL code

Refer to the general instructions in the root `../../../AGENTS.md` for broader instructions.

## Migration scripts SQL code standards

- the code is structured in the following modules
    - `src/main/flyway/sql`: contains the SQL migration scripts for the PostgreSQL database, following Flyway's naming
      conventions and structure
    - `src/main/flyway/conf`: contains the Flyway configuration files
- all SQL migration commands generated by AI must have a comment with the authoring information
